# Transfer

A simple script to upload files to transfer.sh

## Installation

### [zulu](/zulu-zsh/zulu)

```sh
zulu install transfer
```

### Manual

```sh
git clone https://github.com/zulu-zsh/plugin-transfer
chmod u+x transfer
cp transfer /usr/local/bin
```
